# Changelog

Todas as mudanças notáveis do projeto serão documentadas aqui.

O formato segue o [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/).

## [1.0.0] - 2026-03-11

### Adicionado
- **Core**: Sistema de casos (tarefas) com criação, edição e arquivamento
- **Timer**: Timer de 2 minutos para iniciar qualquer tarefa sem desculpa
- **Tribunal**: Julgamento de adiamentos com justificativa obrigatória
- **Modos de jogo**: Padrão, No Escape e Chain (3 tarefas seguidas)
- **Deep Focus**: Sessões de foco profundo de 25 minutos
- **Relatórios**: Dashboard com streak, consistência, coragem e foco
- **Progresso diário**: Tracking de mínimo diário e sistema +1
- **Dívidas**: Sistema de dívidas que custam sessões para quitar
- **Command Line**: Interface via comandos para power users (`Ctrl+K`)
- **Temas**: Sistema de temas customizável com variáveis CSS
- **Perfil**: Perfil de usuário com métricas e login Google
- **Social**: Amigos, leaderboards, ligas e comunidade
- **Co-op**: Salas cooperativas para trabalhar junto
- **Templates**: Templates para criação rápida de casos
- **Atalhos**: Sistema completo de keyboard shortcuts
- **Offline**: Suporte offline com IndexedDB e sync com Firestore
- **Replay**: Visualização semanal de atividades
